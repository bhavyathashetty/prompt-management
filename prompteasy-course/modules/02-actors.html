The data shapes are the contract

A type tells TypeScript what a prompt must contain. This helps catch bad AI-generated changes before they become browser bugs.

CODE 
type Prompt = {
  id: string;
  title: string;
  category: PromptCategory;
  tags: string[];
  content: string;
  favorite: boolean;
  uses: number;
  updatedAt: string;
};
PLAIN ENGLISH

Every prompt has a stable ID so the app can find it later.

Title, category, tags, and content describe the prompt.

Favorite is true or false, like a light switch.

Uses and updatedAt power the dashboard and sorting signals.
